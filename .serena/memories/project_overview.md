Attendance management backend is an ASP.NET Core 10 Web API for a school attendance system. Tech stack: C# 12, .NET 10, ASP.NET Core, EF Core, Identity, SignalR, xUnit, Moq. Structure: attendance_monitoring contains Controllers, IServices, Services, IRepository, Repositories, Classes/DTOs, Exceptions, Constants, Migrations, Program.cs; tests live in attendance.testproject. Architecture follows Controller -> Service -> Repository -> EF Core with services as the main logging boundary.
